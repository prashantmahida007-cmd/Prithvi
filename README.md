# Chess Application

This is a Chess application that allows users to play chess against the computer as well as online with friends. The main features include:

- **Play against AI**: Challenge a smart computer opponent at various difficulty levels.
- **Online Multiplayer**: Connect with friends and play online in real-time.
- **Elegant UI**: Enjoy a stylish and user-friendly interface that enhances the playing experience.
- **Game History**: Keep track of your games with the option to review past matches.

Get ready to strategize and checkmate your opponents!
